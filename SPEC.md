# ClawDeck - Specification

## Tagline
**It's like a fighter jet cockpit, but for your AI sidekicks.**

## What is ClawDeck?
The dashboard for humans who talk to bots—and actually get stuff done. It's what happens when your AI does the dishes AND the laundry. Perfect for bot whisperers, AI enthusiasts, and anyone who wants their digital assistant to feel like PART OF THE TEAM.

## Brand
**ClawDeck** - AI Command Center

## Version
2.0 (Working)

## Features

### Visual Design
- Dark mode UI
- Gradient accents
- Animated lobster logo
- Responsive (mobile-friendly)

### Core Sections
1. **Quote at top** - Daily wisdom
2. **Priorities** (1-4) with dropdowns showing details
3. **Money status** - Income, debt, Canada fund, expenses
4. **Teams** (5) with project dropdowns
5. **To-Do & Accomplishments** - Checklists
6. **Quick Actions** - 8 buttons
7. **Quick Links** - TuneVault, GitHub, etc
8. **Telegram button** - Link to chat
9. **System Status** - Gateway, browser, GitHub
10. **Notes** - Saves to localStorage

### Interactive Features
- Team card hover → dropdown with projects
- Priority hover → dropdown with details
- Notes type → auto-save
- Checkboxes → persist

## Files
- index.html - Complete dashboard
- SKILL.md - Installation guide
- SPEC.md - This file

## Usage
Open index.html in browser or deploy to GitHub Pages

## Customization
Edit index.html to change:
- Name/brand
- Priorities
- Teams
- Links
- Colors
